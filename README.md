# Hulk Playground

Minimal Astro scaffold ready for GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## GitHub Pages Base Path

Set `BASE_PATH` to your repo name if deploying under a project path.

```bash
BASE_PATH=my-repo npm run build
```

Optionally set `SITE_URL` for the canonical site URL.
